📋 Cómo Funciona la Selección Automática

Cuando haces clic en "Editar" un producto, el formulario se llena automáticamente con los datos actuales del producto, incluyendo la categoría.

// En la función editarProducto(id): const producto = allProducts.find(p => p.id === id); if (producto) { // Llenar el formulario con los datos del producto document.getElementById("productName").value = producto.nombre; document.getElementById("productCategory").value = producto.categoria; // ✅ AUTOMÁTICO document.getElementById("productPrice").value = producto.precio; document.getElementById("productStock").value = producto.stock; document.getElementById("productDescription").value = producto.descripcion; // Mostrar el formulario mostrarFormularioAgregar(); }

💡 Nota Importante: La selección automática funciona porque el código JavaScript establece document.getElementById("productCategory").value = producto.categoria; cuando se carga el formulario de edición. Esto asegura que el menú desplegable de categoría muestre la categoría actual del producto.

🌐 Para Probar en el Frontend Real

  1. Abre: http://localhost:3000/productos.html
  2. Busca cualquier producto con categoría definida
  3. Haz clic en el botón "Editar"
  4. ✅ Observa cómo el menú de categoría se selecciona automáticamente
  5. Todos los demás campos también se llenan automáticamente
  6. Puedes modificar cualquier campo y guardar los cambios
