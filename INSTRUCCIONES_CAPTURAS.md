## Especificaciones Técnicas

### Resolución y Calidad
- **Resolución mínima**: 1920x1080 pixels
- **Formato**: PNG (para mejor calidad)
- **Compresión**: Sin pérdida o compresión mínima
- **Tamaño máximo**: 2MB por imagen

### Estilo y Consistencia
- **Navegador**: Chrome o Firefox (última versión)
- **Zoom**: 100% (sin zoom)
- **Tema**: Tema claro del sistema
- **Idioma**: Español

### Captura de Elementos Específicos

#### Formularios
- Incluir campos completos con datos de ejemplo
- Mostrar mensajes de validación
- Capturar estados de error y éxito

#### Modales y Diálogos
- Capturar el modal con el fondo oscurecido
- Incluir botones de acción
- Mostrar diferentes estados si aplica

#### Tablas y Listas
- Incluir encabezados completos
- Mostrar varios registros de ejemplo
- Capturar paginación si aplica

#### Gráficas
- Capturar gráficas con datos reales
- Incluir leyendas y tooltips
- Mostrar interacciones si es posible

## Herramientas Recomendadas

### Para Windows
- **Snipping Tool**: Herramienta nativa de Windows
- **Greenshot**: Software gratuito con opciones avanzadas
- **Lightshot**: Captura rápida y fácil compartir

### Para Mac
- **Cmd + Shift + 4**: Captura de selección
- **Skitch**: Anotaciones y edición
- **Monosnap**: Captura con grabación

### Para Todos
- **Nimbus Capture**: Extensión de navegador
- **Awesome Screenshot**: Extensión Chrome/Firefox
- **ShareX**: Captura y edición avanzada

## Proceso de Captura

### 1. Preparación
1. Iniciar sesión en el sistema con datos de prueba
2. Asegurar que todos los datos sean consistentes
3. Limpiar caché del navegador
4. Ajustar resolución a 1920x1080

### 2. Captura
1. Abrir la página específica
2. Realizar las acciones necesarias
3. Capturar en el momento adecuado
4. Guardar con nombre descriptivo

### 3. Edición (Opcional)
1. Recortar si es necesario
2. Añadir flechas o números si ayuda
3. Redimensionar si excede el tamaño máximo
4. Optimizar para web

### 4. Organización
1. Guardar en la carpeta correspondiente
2. Usar nombres consistentes
3. Mantener un registro de capturas realizadas
4. Actualizar el manual con las nuevas imágenes

## Checklist de Calidad

### Antes de Capturar
- [ ] Sistema funcionando correctamente
- [ ] Datos de prueba cargados
- [ ] Navegador en 100% zoom
- [ ] Resolución adecuada
- [ ] Sin elementos personales visibles

### Durante la Captura
- [ ] Elemento completo visible
- [ ] Sin cortes importantes
- [ ] Buena iluminación/contraste
- [ ] Texto legible
- [ ] Elementos interactivos visibles

### Después de Capturar
- [ ] Nombre descriptivo
- [ ] Formato correcto (PNG)
- [ ] Tamaño adecuado
- [ ] Calidad suficiente
- [ ] Ubicación correcta en carpeta

## Integración con el Manual

### Actualización del Manual
1. Reemplazar marcadores de posición con rutas reales
2. Verificar que todas las imágenes carguen correctamente
3. Probar el manual en diferentes dispositivos
4. Actualizar números de figura si es necesario

### Mantenimiento
- Revisar capturas cada actualización mayor del sistema
- Actualizar imágenes cuando cambie la interfaz
- Mantener backup de las capturas originales
- Documentar cambios en las imágenes
