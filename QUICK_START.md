# ⚡ INICIO RÁPIDO - Systemsware

## El error que ves significa:
```
Error de conexión con el servidor
❌ No se pudo conectar al servidor
```

**Solución: Necesitas iniciar el servidor Node.js**

---

## 3 PASOS RÁPIDOS ✨

### PASO 1️⃣: Abre una terminal en esta carpeta
- Haz clic derecho en esta carpeta
- Selecciona "Abrir terminal aquí" o "Abrir PowerShell aquí"

### PASO 2️⃣: Instala dependencias (solo la primera vez)
```
npm install
```

### PASO 3️⃣: Inicia el servidor
```
npm start
```

**Deberías ver:**
```
Server running on port 3000
```

---

## ✅ ¿Ya funciona?
- Abre tu navegador
- Ve a: http://localhost:3000
- ¡Listo! 🎉

---

## 🐛 Si algo falla:

**"No se encuentra npm"**
→ Instala Node.js desde: https://nodejs.org/

**"Error al conectar a base de datos"**
→ Asegúrate que PostgreSQL está ejecutándose

**"Puerto 3000 en uso"**
→ Edita `server.js` línea 10 y cambia `3000` por `3001`

---

## 📁 Alternativa: Usa el archivo .bat
- Haz doble clic en `iniciar-servidor.bat`
- Se abrirá una terminal y iniciará automáticamente
