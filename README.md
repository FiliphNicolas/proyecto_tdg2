# proyecto_tdg2

## Descripción

Systemsware es un sistema de inventario web diseñado para pequeñas y medianas empresas que requieren un control fiable y trazabilidad de sus existencias, movimientos y usuarios. La aplicación centraliza la gestión de productos, almacenes, entradas y salidas de stock, y proporciona herramientas para generar reportes detallados sobre ventas, ajustes e histórico de inventario.

La plataforma incluye módulos para registro e inicio de sesión de usuarios con roles (administrador, operador, lector), gestión completa del catálogo de productos (categorías, proveedores, códigos y descripciones), control de stock por ubicación y alertas por bajo inventario. Además cuenta con generación de reportes exportables, un sistema de soporte/chat integrado y pantallas para administración de perfiles y configuración del sistema.

Principales características:

- Registro e inicio de sesión con roles y permisos.
- Gestión de productos: alta, baja, modificación, categorías y proveedores.
- Control de inventario por ubicaciones y movimientos (entradas/salidas/ajustes).
- Generación de reportes y exportación de datos para auditoría.
- Interfaz de soporte/chat para asistencia interna.
- Páginas de administración de usuarios y perfiles.

Diseñado para ser intuitivo y fácil de desplegar, `Systemsware` facilita la toma de decisiones mediante datos confiables y búsquedas rápidas, reduciendo pérdidas por errores de control y mejorando la eficiencia operativa.
